# 15 JavaScript Projects

A collection of 15 mini projects built with **vanilla JavaScript**, covering DOM manipulation, events, and interactive UI logic.  
Based on [John Smilga’s “Build 15 JavaScript Projects” course](https://www.youtube.com/watch?v=3PHXvlpOkf4) from [freeCodeCamp.org](https://www.freecodecamp.org/).

---

## Purpose
To strengthen fundamental **JavaScript (ES6+)** skills by building functional, interactive projects **without frameworks** — focusing on clean logic, event handling, and DOM updates.

---
